dotfiles
--------

## Getting started
First, you must clone the repository in your development computer.

```sh
$ git clone https://github.com/pine613/dotfiles.git
```

### Linux

```sh
$ ./bin/install.bash # install .files
$ ./bin/install.bash --deps # install anyenv, NeoBundle, etc ...
```
