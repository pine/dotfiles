## Getting started
First, you must clone the repository in your development computer.

```sh
$ git clone https://github.com/pine613/dotfiles.git
```

### Linux

```sh
$ ./bin/install.bash
```

### Windows

```sh
$ .\bin\install.bat
```

## Test
### Linux

```sh
$ npm install
$ npm test
```
